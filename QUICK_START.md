# ⚡ Inicio Rápido - Chat Global

## 🎯 Empezar en 3 Pasos

### Paso 1: Abrir la Aplicación

**Opción A - Localmente:**
```bash
# Simplemente abre index.html en tu navegador
# Doble click en el archivo o:
open index.html  # Mac
start index.html # Windows
xdg-open index.html # Linux
```

**Opción B - Con servidor local:**
```bash
# Python
python -m http.server 8000

# Node.js
npx http-server

# Luego abre: http://localhost:8000
```

### Paso 2: Configurar tu Perfil

1. Selecciona tu idioma (ej: Español 🇪🇸)
2. Escribe tu nombre (ej: Diego)
3. Copia tu ID único (aparece arriba)

### Paso 3: Conectar

**Para conectarte con alguien:**

1. **Tú compartes tu ID** → Envíalo por WhatsApp/Telegram
2. **Recibes el ID de tu amigo** → Pégalo en "ID del Amigo"
3. **Click en "Conectar"** → ¡Ya están chateando!

## 💬 Ejemplo de Uso

### Escenario: Quieres hablar con un japonés

1. **Tú (Español)**:
   - Idioma: 🇪🇸 Español
   - Tu ID: `abc123xyz` (ejemplo)
   
2. **Tu amigo (Japonés)**:
   - Idioma: 🇯🇵 日本語
   - Su ID: `def456uvw` (ejemplo)

3. **Conectarse**:
   ```
   Tú envías: "abc123xyz" a tu amigo
   Tú pegas: "def456uvw" en tu app
   Click: "🔗 Conectar"
   ```

4. **Activar traducción**:
   - Click en el botón 🌐 arriba del chat
   - Ahora tus mensajes se traducen automáticamente

5. **Chatear**:
   ```
   Tú escribes: "Hola, ¿cómo estás?"
   Tu amigo ve: "こんにちは、元気ですか？"
   
   Él escribe: "元気です！"
   Tú ves: "¡Estoy bien!"
   ```

## 🛠️ Solucionar Problemas Comunes

### "No aparece mi ID"
✅ Espera 3-5 segundos al cargar la página
✅ Refresca la página (F5)
✅ Revisa tu conexión a internet

### "Usuario no disponible"
✅ Ambos deben estar conectados AL MISMO TIEMPO
✅ Verifica que el ID sea exacto (sin espacios)
✅ Tu amigo debe tener la página abierta

### "Los mensajes no llegan"
✅ Revisa tu conexión a internet
✅ Ambos deben tener la ventana del chat abierta
✅ Intenta desconectar y reconectar

### "La traducción no funciona"
✅ Activa el botón 🌐
✅ Verifica tu conexión a internet
✅ La traducción usa un servicio externo (puede tardar 1-2 segundos)
